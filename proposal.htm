Project Topic

I plan to create a simple scene like the one pictured below that includes one or more textured objects that can be reflected in a working planar mirror. GLSL shaders will be used to apply lighting and create the reflection. I will make it possible to rotate the mirror and move one of the objects around the scene using the keyboard.

Design of Application

  • Based off of code from previous labs
  • Fixed camera position
  • Phong shading on all objects from a directional light
  • Textured ground plane
  • Textured object like pictured vase that can be moved around surface of ground plane
  • Mirror that can rotate about fixed point
  • Reflection will (probably) be generated as follows:
    1. Render reflected geometry based on current orientation of mirror
    2. Clip scene at plane of mirror
    3. Store image into a texture
    4. Clear
    5. Render entire scene, using stored texture for mirror surface
  • I may decide to do reflection using stencil erasing instead of texturing the mirror surface

Timeline

  1. Render geometry of entire scene
  2. Texture all objects except mirror
  3. Generate reflection
  4. Lighting of both regular and reflected objects
  5. If time permits, I will add a large textured sphere for background
