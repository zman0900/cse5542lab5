Project Topic

I have created a simple scene like the one pictured below that includes one or more textured objects that can be reflected in a working planar mirror. GLSL shaders are used to apply lighting and textures. It is possible to rotate the mirror and move one of the objects around the scene using the keyboard.

Design of Application

  • Based off of code from previous labs
  • Fixed camera position, allowing zoom
  • Vertex Buffer Objects and Vertex Array Objects are used to pass all vertex data to GLSL shaders for texturing and lighting
  • Multisampling is enabled. Toggle with the m key
  • Dog
    • Phong shading from a directional light
    • Textured with targa image
    • Also uses a specular map from a targa image
    • Movable using the i,j,k,l keys
    • Rotate with the u,o keys
    • Dog was used instead of vase because no good detailed vase models could be found
  • Ground plane
    • Textured with a procedurally generated texture
  • Background
    • Large textured sphere from lab 4 was reused.
    • Loads HDR (floating point) texture if the hardware supports it, otherwise falls back to integer
  • Mirror
    • Able to rotate about a fixed point using the < and > keys
    • Texture for the mirror is generated using a Frame Buffer Object:
      1. Switch rendering to FBO
      2. Set viewport to match dimensions of desired texture
      3. Transform so mirror is at origin and lined up with the x,y plane
      4. Reflect across the z plane
      5. Perform the inverse of the previous transformations
      6. Render everything in the scene except for plane of the mirror
      7. Switch back to the normal frame buffer
      8. Render mirror using texture from the FBO
      9. Render everything else in the scene again

Using the Application

A list of controls is provided in the output to the terminal when the application is run.

    Linux:
  • cmake .
  • make
  • ./lab5
    Windows:
  • Open the cse5542lab5.sln file in Visual Studio 2012
  • Ensure the your Visual Studio install has access to glut (or freeglut), glew32, and the opengl libs
  • Build and run
